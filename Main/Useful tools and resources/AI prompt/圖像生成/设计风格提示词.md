**一、现代潮流风格**（适合互联网/年轻群体）

1. 酸性设计  
    • 特征：金属流体、霓虹光效、几何镂空  
    • 识别口诀："像融化的金属糖纸"  
    • 示例：铬银液体＋荧光绿边框＋网格透视  
    
2. 赛博朋克  
    • 特征：霓虹灯管、全息投影、机械元件  
    • 识别口诀："香港夜市＋黑客帝国"  
    • 示例：品红＋青蓝撞色、电子雨效果  
    
3. 3D磨砂玻璃  
    • 特征：半透明模糊、背景虚化、光晕  
    • 识别口诀："毛玻璃手机图标"  
    • 示例：浅蓝雾面按钮＋彩色弥散投影  
    
4. Y2K千禧风  
    • 特征：镭射材质、低保真像素、糖果色  
    • 识别口诀："翻盖手机时代审美"  
    • 示例：粉紫渐变＋数码雪花噪点  
    
5. 新丑风  
    • 特征：混乱排版、高饱和撞色、粗糙笔触  
    • 识别口诀："故意做得很土很扎眼"  
    • 示例：红配绿＋扭曲文字＋手绘涂鸦  
    

  

**二、经典艺术风格**（适合文化/高端场景）

1. 包豪斯  
    • 特征：几何构成/无衬线体/红黄蓝原色  
    • 识别口诀："蒙德里安格子画"  
    • 示例：黑色直线分割+三原色色块  
    
2. 超现实主义  
    • 特征：梦境场景/物体异化/空间错位  
    • 识别口诀："达利的融化时钟"  
    • 示例：悬浮山+巨手托举建筑  
    
3. 波普艺术  
    • 特征：网点印刷商业符号重复排列  
    • 识别口诀："安迪·沃霍尔的罐头"  
    • 示例：玛丽莲梦露头像四联拼贴  
    
4. 哥特风  
    • 特征：尖拱结构/宗教纹样/暗黑系  
    • 识别口诀："吸血鬼城堡既视感"  
    • 示例：血红色玫瑰+铁艺纹章  
    
5. 中国水墨  
    • 特征：飞自笔触/留白意境/黛青色  
    • 识别口诀："山水画裱在卷轴上"  
    • 示例：毛笔字题词+远山淡影  
    

  

**三、功能导向风格**（适合商业/产品设计）

1. 拟物化  
    • 特征：真实质感/细节阴影/立体按钮  
    • 识别口诀："像实物照片"  
    • 示例：皮革缝线+金属铆钉效果  
    
2. 极简扁平  
    • 特征：无阴影/纯色块/极细线框  
    • 识别口诀："iOS系统图标"  
    • 示例：纯色圆形+单线图标  
    
3. MBE插画  
    • 特征：粗描边/溢出色块/小元素点缀  
    • 识别口诀："断线的小太阳笑脸"  
    • 示例：黑色轮廓线+粉蓝撞色  
    
4. 等距投影  
    • 特征：2.5D视角/统一角度/无透视  
    • 识别口诀："乐高积木式立体"  
    • 示例：倾斜30°的城市模型  
    
5. 数据可视化  
    • 特征：信息图表/箭头指引/渐变色阶  
    • 识别口诀："学术论文配图"  
    • 示例：饼状图+上升趋势折线  
    

  

**四、情绪化风格**（适合营销/社交媒体）

1. 蒸汽波  
    • 特征：故障网格/石膏像/日文字  
    • 识别口诀："Vindows98怀旧屏保"  
    • 示例：粉紫渐变+罗马柱残片  
    
2. 故障艺术  
    • 特征：颜色分离/条纹干扰/像素错位  
    • 识别口诀："损坏的DVD效果"  
    • 示例：RGB通道错位+扫描线  
    
3. 手帐拼贴  
    • 特征：胶带纸/便签条/手写字体  
    • 识别口诀："少女日记本内页"  
    • 示例：牛皮纸底+邮票式贴图  
    
4. 森系治愈  
    • 特征：手绘植物/浅色系/棉麻质感  
    • 识别口诀："无印良品海报"  
    • 示例：水彩树叶+亚麻布纹理  
    
5. 毛绒质感  
    • 特征：绒毛边缘/柔软渐变/温暖色调  
    • 识别口诀："摸起来像云朵"  
    • 示例：奶黄色+棉花糖字体
