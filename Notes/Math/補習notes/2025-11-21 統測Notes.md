#### **1. 代數方式（列方程）**
**核心口訣：** **把文字翻譯成數學式子**
- **要領：**
    1. **定義未知數 (Let x be...):** 清楚地設「誰」是 x，通常問什麼就設什麼。
    2. **尋找等量關係：** 這是解題的關鍵。題目中一定有像「相等」、「比...多/少」、「總和」這樣的詞語，這就是等號  =  的位置。
    3. **依等量關係列方程：** 用 x 和已知數把等號兩邊的式子寫出來。
    4. **解方程並驗算：** 解出 x 後，**一定要代回題目**檢查是否符合題意。

#### **2. 連比**
**核心口訣：** **找到共同橋樑，化為最簡整數比**
- **要領：**
    1. **引入未知數 k：** 若 a : b : c = 2 : 3 : 5，則可設 `a = 2k`, `b = 3k`, `c = 5k`。這個技巧可以將比例問題轉為代數問題，非常好用。
    2. **找出「公倍數」橋樑：** 如果 a : b = 2 : 3, b : c = 6 : 5，要連起來就必須讓兩個比中的「b」值相同。3 和 6 的最小公倍數是 6，所以將第一個比乘以 2 → a : b = 4 : 6，就能得到 a : b : c = 4 : 6 : 5。
    3. **應用題：** 常用於分配問題（例如按比例分錢）。先求總份數（連比相加），再計算每一份的值。
        

#### **3. 百分數**
**核心口訣：** **認清「百份之幾」的「基數」是誰**
- **要領：**
    1. **三大類問題：**
        - **求 P% 是多少：** `基數 × P%`
        - **求 A 是 B 的百分之幾：** `(A / B) × 100%`
        - **已知某數的 P% 是 C，求某數：** `某數 = C ÷ P%`
    2. **百分增加/減少：**
        - **新值 = 原值 × (1 + 增加%)** 或 **新值 = 原值 × (1 - 減少%)**
        - **變化率 = (新值 - 原值) / 原值 × 100%**
    3. **盈利與虧蝕：** 成本是 100%，售價是 `成本 × (1 + 盈利%)` 或 `成本 × (1 - 虧蝕%)`。
        
#### **4. 誤差**
**第一步：確定「準確至」哪一個數位**
- **準確至某位有效數字**
- **例句：** 「123.4 準確至 4 位有效數字」
- **判斷方法：** 從第一個非零數字開始數，數到最後一位。
    - `3.60` (3位有效數字，準確至0.01)
    - `0.0800` (3位有效數字，準確至0.0001)
    - `49000` (2位有效數字，準確至1000)

#### **5. 體積**
- 角柱/圓柱體積 = **底面積 × 高**
- 角錐/圓錐體積 = **(1/3) × 底面積 × 高**
- 球體體積 = **(4/3) × π × r³**

#### **6. 不等式**
**核心口訣：** **解法和方程類似，但乘除負數要「反向」**
- **要領：**
    1. **解不等式：** 和解方程一樣，用加減乘除移項。
    2. **唯一關鍵：** 當不等式**兩邊同時乘或除以一個負數**時，不等號的方向必須**反轉**！(`>` 變 `<`, `≤` 變 `≥`) 這是最大失分點！
    3. **在數線上表示解：**
        - `>` 或 `<`：用空心圈 `○`
        - `≥` 或 `≤`：用實心點 `●`
    4. **文字題：** 讀懂題目，找出「超過」、「至少」、「不超過」這些關鍵詞，轉成不等式符號 (`>`, `≥`, `≤`)。
        

#### **7. 圓的性質**
**對題目一條一條仔細看，每一句都是綫索**
1. **五大必考定理（要會看圖說出名稱）：**
	- **圓心角是圓周角兩倍**
	- **同弓形內的圓周角相等**
	- **半圓上的圓周角是直角** 
	- **對角互補** 
	- **弦切角等於對應內對角**
2. **解題步驟：**
	- **Step 1：** 在圖上標出所有已知的角。
	- **Step 2：** 尋找「半圓」、「等腰三角形」（因半徑相等）、「直徑」、「切綫」等特殊圖形。
	- **Step 3：** 用已知角一個接一個地推算出其他未知角。
